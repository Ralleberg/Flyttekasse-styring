SERVER-ONLY MAMP VERSION

1) Copy the folder 'flyttekasser' to:
   /Applications/MAMP/htdocs/

2) Start MAMP (Apache)

3) Open:
   http://localhost:8888/flyttekasser/

Endpoints:
- http://localhost:8888/flyttekasser/load.php
- http://localhost:8888/flyttekasser/save.php
- http://localhost:8888/flyttekasser/data.json
